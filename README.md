# property-photo-tracker

A web application that will allow users to enter their portal to track and manage the photos they take at property damage insurance claims. Those photos that are uploaded to the web app will be the ones analyzed by the back end API for analysis and that analysis will be provided to the end user

## Setup

```sh

cd property-photo-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
